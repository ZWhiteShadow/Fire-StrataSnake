# Fire StrataSnake

## Overview

Fire StrataSnake is a strategy-based Snake game developed for Android and Amazon Fire TV.

Rather than recreating the traditional Snake experience, this project expands the classic gameplay with progression mechanics, score multipliers, persistent high scores, and strategic decision making. The goal was to build a larger Java application while gaining experience with Android development, object-oriented programming, and game architecture.

The project also provided experience designing software for television interfaces and controller-based gameplay.

---

## Key Features

- Classic Snake gameplay with strategic enhancements
- Progressive gameplay difficulty
- Score multiplier system
- Persistent high-score tracking
- Controller-friendly interface
- Designed for Android and Amazon Fire TV
- Custom game logic and progression mechanics

---

## Technologies

- Java

## Development Process

This project focused on expanding a familiar game into a larger object-oriented application.

Development included designing reusable game components, implementing movement and collision systems, managing game state, persisting player scores, and balancing gameplay to create a more engaging experience than a traditional Snake implementation.

Building the project also provided hands-on experience with Android application development and adapting gameplay for television devices and remote controls.
